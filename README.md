# Elegant Dark Stopwatch

[![Netlify Status](https://api.netlify.com/api/v1/badges/d1595cde-eb54-485d-bf31-bdef37fdd25c/deploy-status)](https://app.netlify.com/projects/ezwatch/deploys)

[**Live Demo**](https://ezwatch.netlify.app/)


A premium, dark-themed stopwatch application featuring a modern glassmorphism UI, smooth animations, and a focus on user experience.

## Features

-   **Precision Timing**: Tracks minutes, seconds, and milliseconds.
-   **Glassmorphism UI**: Elegant, translucent design with animated background blobs.
-   **Full Screen Mode**: Toggle full-screen for an immersive experience.
-   **Keyboard Shortcuts**:
    -   `Enter`: Start / Stop (Reset)
    -   `Space`: Pause / Resume
-   **Onboarding Overlay**: Interactive guide highlighting controls on first visit.
-   **Responsive Design**: Optimized for various screen sizes with responsive arrow alignment.
-   **Secure Implementation**: Logic encapsulated in an IIFE to prevent console tampering.
-   **Custom Typography**: Uses 'Roboto Mono' for the timer and 'Fira Code' for UI elements.

## Tech Stack

-   **HTML5**: Semantic structure.
-   **CSS3**: Advanced styling with variables, animations, and flexbox/grid.
-   **JavaScript (Vanilla)**: Core logic with event handling and DOM manipulation.

## Setup & Usage

1.  Clone the repository or download the source code.
2.  Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3.  Enjoy the stopwatch!

## Controls

| Action | Button Icon | Keyboard Shortcut |
| :--- | :---: | :---: |
| **Start** | `play_arrow` | `Enter` |
| **Pause** | `pause` | `Space` |
| **Resume** | `play_arrow` | `Space` |
| **Stop / Reset** | `stop` | `Enter` |
| **Full Screen** | `fullscreen` | N/A |
